Native Launcher (all in bin)
----------------------------
The native launcher is compiled with Cygwin support and for 32-bit environments.
In Supplemental you find a directory "NativeLauncherWithoutCygwinSupport" that 
contains versions of the native launcher compiled without the Cygwin support. If
you have any problems with the native launcher please try these versions.
Currently the native launcher cannot be used for 64-bit installations of Java,
since there seems to be nobody able to do the compilation for 64-bit.

64-Bit Versions of Java
-----------------------
The installer tries to find out whether a 32-bit or a 64-bit version of Java is installed.
If the installer cannot determine this a 32-bit installation is assumed. The difference 
for the time being is that for the 64-bit version no native launcher is installed. As soon
as a 64-bit version of the native launcher becomes available, the installer will use that
version of the launcher.
